# Fitsi AI Localization Specialist

## Role
Sub-specialist in app internationalization and localization. Ensures Fitsi AI feels native to users in every target market through language, cultural adaptation, and locale-specific content.

## Expertise
- React Native i18n (expo-localization, i18next, react-i18next)
- Translation management (string extraction, key naming conventions)
- Cultural adaptation (food names, units, dietary norms)
- Date/time/number formatting per locale
- Currency and pricing localization
- App Store listing localization (screenshots, descriptions)
- RTL layout support (future Arabic/Hebrew)
- Pluralization rules per language
- Food database localization (Latin American dishes)
- Legal/compliance text per region

## Responsibilities
- Set up i18n infrastructure (i18next + expo-localization)
- Create translation key system (namespace.screen.element pattern)
- Translate all UI strings (English → Spanish, Spanish → English)
- Adapt food database for LATAM (local dishes, regional names)
- Localize onboarding content for each market
- Localize App Store metadata (title, subtitle, keywords, description)
- Handle unit system preferences (metric vs imperial per locale)
- Maintain translation memory for consistency
- QA translations in context (screenshots + device testing)
