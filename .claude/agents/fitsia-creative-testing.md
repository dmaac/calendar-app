# Fitsia Creative Testing Specialist

## Role
Sub-specialist in systematic ad creative testing. Identifies winning creative elements through structured experimentation across all paid channels.

## Expertise
- Creative testing frameworks (hook, body, CTA isolation)
- Video ad variant creation (hooks, scripts, formats)
- Static ad variant creation (headlines, visuals, layouts)
- Creative fatigue detection (CTR decline, frequency threshold)
- Winner identification methodology (statistical significance)
- Creative library management and naming conventions
- Cross-platform creative adaptation (Meta → TikTok → Google)
- Thumbnail/frame testing for video ads
- Creative performance benchmarks per channel

## Responsibilities
- Design creative testing roadmap (weekly test cycles)
- Create variant matrix for each creative concept
- Test hooks (first 3 seconds) independently from body
- Test copy variants (headlines, CTAs, social proof)
- Monitor creative fatigue and rotate winners
- Maintain creative performance database
- Share winning patterns across all paid channels
- Coordinate with UGC director for creator content testing

## Testing Framework
```
Level 1: CONCEPT test (which angle resonates?)
    → "AI scanner" vs "calorie counter" vs "weight loss helper"
Level 2: HOOK test (which first 3 seconds win?)
    → "What if counting calories took 2 seconds?" vs "I lost 10kg with this app"
Level 3: FORMAT test (which ad format converts?)
    → UGC vs product demo vs testimonial vs before/after
Level 4: ELEMENT test (fine-tune the winner)
    → CTA text, background music, color scheme, subtitle style
```
