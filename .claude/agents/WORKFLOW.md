## Flujos de Trabajo Principales

### WORKFLOW A — Feature Development Sprint
```
product-manager (user story)
  → ux-researcher (validar con personas)
  → project-coordinator (asignar sprint task)
  → ui-engineer / onboarding-builder (build screen)
  → python-backend-engineer (API endpoint)
  → api-contract-guardian (alinear tipos FE↔BE)
  → data-migration-agent (schema si aplica)
  → qa-engineer (tests)
  → senior-code-reviewer (review)
  → devops-deployer (deploy)
  → ux-polish-agent (animations & polish)
```

### WORKFLOW B — AI Food Scan Pipeline
```
nutrition-science-advisor (precision requirements)
  → ai-vision-expert (build GPT-4o/Claude vision pipeline)
  → python-backend-engineer (POST /api/food/scan endpoint)
  → data-migration-agent (ai_scan_cache schema)
  → api-contract-guardian (TypeScript types)
  → health-data-scientist (feedback loop para mejorar accuracy)
  → health-compliance-agent (disclaimers requeridos)
```

### WORKFLOW C — Launch Readiness
```
fullstack-inspector (full audit)
  → security-engineer (security audit)
  → health-compliance-agent (App Store compliance)
  → qa-engineer (test coverage report)
  → scalability-architect (infra review)
  → devops-deployer (production deploy)
```

### WORKFLOW D — Marketing Campaign Launch
```
competitor-analyst (market intel)
  → growth-strategist (strategy + budget allocation)
    ├── aso-specialist + aso-copywriter → App Store organic
    ├── ugc-content-director → creator briefs (→ Meta + TikTok)
    ├── meta-ads-specialist → Facebook/Instagram campaigns
    ├── tiktok-ads-specialist + ugc-content-director → TikTok
    ├── apple-search-ads-specialist → ASA
    ├── google-uac-specialist → Google UAC
    ├── influencer-partnership-manager → creator partnerships
    ├── email-funnel-builder → lifecycle + push notifications
    └── cro-landing-page-specialist → web-to-app funnel
  → paid-analytics-specialist (measure ALL channels)
  → retention-growth-specialist (post-install retention)
  → data-analyst (report back → growth-strategist loop)
```

### WORKFLOW E — Data & Optimization Loop (semanal)
```
data-analyst (cohort analysis D1/D7/D30)
  → health-data-scientist (ML churn model improvements)
  → paid-analytics-specialist (CAC/LTV actualizados por canal)
  → retention-growth-specialist (retention tactics ajustadas)
  → growth-strategist (rebalance budget & strategy)
  → product-manager (product improvements from data)
```
