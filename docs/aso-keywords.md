## Keyword Review Cadence

- **Weekly (first month):** Check rankings via AppTweak/Sensor Tower. Adjust keyword field based on indexing.
- **Bi-weekly (months 2-3):** Swap underperforming keywords. A/B test subtitle variants.
- **Monthly (ongoing):** Full keyword audit. Add seasonal terms (New Year's resolution, summer body, etc.).
- **Quarterly:** Competitor keyword gap analysis. Update long-tail strategy.

## Metrics to Track

- **Keyword rankings** (target: top 10 for primary, top 30 for secondary within 90 days)
- **Search visibility score** (AppTweak/Sensor Tower)
- **Conversion rate per keyword** (impressions to installs)
- **Organic install share** (vs. paid)
- **Keyword install volume** (which keywords drive actual downloads)
